# 📥 下载与安装

## 方式一：从 GitHub Releases 下载（推荐）

### 步骤 1：访问 Release 页面

```
https://github.com/YOUR_USERNAME/BluetoothMusicApp/releases
```

### 步骤 2：下载 APK

点击最新版本（v1.0.0）下的附件：
- **app-debug.apk** - 调试版（推荐测试使用）
- **app-release.apk** - 发布版（正式使用）

### 步骤 3：传输到手机

**方法 A：USB 传输**
1. 用 USB 线连接手机和电脑
2. 将 APK 文件复制到手机
3. 在手机上点击 APK 安装

**方法 B：无线传输**
- 微信/QQ 文件传输助手
- 电子邮件附件
- 云盘（Google Drive、Dropbox、百度网盘等）

### 步骤 4：安装

1. 在手机上找到 APK 文件
2. 点击安装
3. 如果提示"未知来源"，允许安装
4. 等待安装完成

## 方式二：自行构建

### 前提条件

- Android Studio Arctic Fox 或更新版本
- JDK 11 或更高版本
- Android SDK

### 步骤 1：克隆仓库

```bash
git clone https://github.com/YOUR_USERNAME/BluetoothMusicApp.git
cd BluetoothMusicApp
```

### 步骤 2：用 Android Studio 打开

1. 启动 Android Studio
2. `File → Open`
3. 选择 `BluetoothMusicApp` 文件夹
4. 等待 Gradle 同步完成

### 步骤 3：配置 SDK 路径

编辑 `local.properties` 文件：

**macOS:**
```
sdk.dir=/Users/你的用户名/Library/Android/sdk
```

**Windows:**
```
sdk.dir=C\:\\Users\\你的用户名\\AppData\\Local\\Android\\Sdk
```

**Linux:**
```
sdk.dir=/home/你的用户名/Android/Sdk
```

### 步骤 4：构建 APK

**方法 A：Android Studio**
```
Build → Build Bundle(s) / APK(s) → Build APK(s)
```

**方法 B：命令行**
```bash
# macOS/Linux
chmod +x gradlew
./gradlew assembleDebug

# Windows
gradlew.bat assembleDebug
```

### 步骤 5：找到 APK

```
app/build/outputs/apk/debug/app-debug.apk
```

## 📱 安装后设置

### 1. 首次启动

打开应用，授予以下权限：
- ✅ 蓝牙权限
- ✅ 通知权限（Android 13+）

### 2. 开启无障碍权限（重要！）

这是实现 **BGM→播放列表自动切换** 的关键！

**设置路径：**
```
设置 → 无障碍 → 已下载的服务 → 蓝牙音乐控制 → 开启
```

**不同品牌手机：**
- **小米**: 设置 → 更多设置 → 无障碍
- **华为**: 设置 → 辅助功能 → 无障碍
- **OPPO**: 设置 → 系统设置 → 无障碍
- **vivo**: 设置 → 快捷与辅助 → 无障碍
- **三星**: 设置 → 辅助功能

### 3. 关闭电池优化

```
设置 → 应用 → 蓝牙音乐自动化 → 电池 → 允许后台活动
```

### 4. 配置自动化

1. 选择蓝牙设备
2. 选择 Apple Music
3. 设置 BGM 歌曲（如：Welcome Home）
4. 设置播放列表（如：我的最爱）
5. 选择播放模式
6. 保存配置

### 5. 测试

1. 断开蓝牙连接
2. 重新连接
3. 观察是否自动播放音乐

## ⚠️ 常见问题

### Q1: 安装时提示"未知来源"？

**解决方案：**
- 允许"未知来源"安装
- 设置 → 安全 → 未知来源 → 允许

### Q2: 应用无法打开？

**解决方案：**
- 检查 Android 版本（需要 8.0+）
- 重新安装
- 查看日志：`adb logcat | grep BluetoothMusic`

### Q3: 找不到蓝牙设备？

**解决方案：**
- 先在系统设置中配对蓝牙设备
- 重启应用
- 检查蓝牙权限

### Q4: 音乐不自动播放？

**解决方案：**
- 确保已开启无障碍权限
- 确保 Apple Music 已安装
- 手动打开一次 Apple Music
- 检查后台服务是否在运行

### Q5: 应用被系统杀死？

**解决方案：**
- 关闭电池优化
- 允许自启动
- 锁定应用到最近任务
