## 🚀 发布到 GitHub（3 种方式）

### 方式一：使用发布脚本（最简单）

```bash
# 进入项目目录
cd /home/admin/openclaw/workspace/BluetoothMusicApp

# 运行发布脚本
./publish-to-github.sh
```

脚本会：
1. 检查 Git 仓库状态
2. 提示你输入 GitHub 仓库 URL
3. 推送代码到 GitHub
4. 显示下一步操作指引

---

### 方式二：手动命令行

#### 步骤 1：在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 仓库名称：`BluetoothMusicApp`
3. 描述：`🎵 安卓蓝牙音乐自动化应用 - 支持 Apple Music BGM→播放列表自动切换`
4. 设为 **Public** 或 **Private**（你的选择）
5. **不要**添加 README/.gitignore/License
6. 点击 `Create repository`

#### 步骤 2：推送代码

```bash
cd /home/admin/openclaw/workspace/BluetoothMusicApp

# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/BluetoothMusicApp.git

# 推送
git push -u origin main
```

**如果需要认证：**
- 用户名：你的 GitHub 用户名
- 密码：[Personal Access Token](https://github.com/settings/tokens)（不是账号密码）

---

### 方式三：GitHub Desktop

1. 下载 https://desktop.github.com/
2. 打开 GitHub Desktop
3. `File → Add Local Repository`
4. 选择 `BluetoothMusicApp` 文件夹
5. 点击右上角 `Publish repository`
6. 完成！

## 📥 构建 APK 并上传

### 在 Android Studio 中构建

1. 用 Android Studio 打开项目
2. `Build → Build APK(s)`
3. 等待构建完成
4. APK 位置：`app/build/outputs/apk/debug/app-debug.apk`

### 上传到 Release

1. 访问 `https://github.com/YOUR_USERNAME/BluetoothMusicApp/releases/new`
2. Tag version: `v1.0.0`
3. Release title: `v1.0.0 - 初始版本`
4. 拖拽 APK 文件到上传区域
5. 点击 `Publish release`

## 🔑 Personal Access Token 创建

如果推送时需要认证：

1. 访问 https://github.com/settings/tokens
2. 点击 `Generate new token (classic)`
3. 名称：`BluetoothMusicApp`
4. 过期时间：`90 days`
5. 勾选权限：`repo` (Full control)
6. 点击 `Generate token`
7. **复制 Token**（只会显示一次）
8. 推送时使用 Token 作为密码
