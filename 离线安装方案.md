## 由于网络限制，推荐以下方案：

---

## 方案一：本地构建 APK（推荐）

### 步骤 1：复制项目到本地电脑

**Windows:**
```powershell
# 使用 SCP 或文件传输工具
# 将整个 BluetoothMusicApp 文件夹复制到电脑
```

**macOS/Linux:**
```bash
# 使用 SCP
scp -r admin@server:/home/admin/openclaw/workspace/BluetoothMusicApp ~/Desktop/
```

**或者：**
- 使用文件传输工具（WinSCP、FileZilla 等）
- 打包成 ZIP 下载

---

### 步骤 2：用 Android Studio 构建 APK

1. **打开 Android Studio**
2. **File → Open** → 选择 `BluetoothMusicApp` 文件夹
3. **等待 Gradle 同步完成**
4. **Build → Build APK(s)**
5. **等待构建完成**

APK 输出位置：
```
app/build/outputs/apk/debug/app-debug.apk
```

---

### 步骤 3：安装到手机

**方法 A：USB 直连**
```bash
# 手机开启 USB 调试
# 连接 USB 线
adb install app/build/outputs/apk/debug/app-debug.apk
```

**方法 B：文件传输**
1. 将 APK 发送到手机（微信/QQ/邮件/数据线）
2. 在手机上点击 APK 安装
3. 允许"未知来源"

## ⚠️ 注意事项

### 1. Android SDK 配置

首次打开项目时，编辑 `local.properties`：

**Windows:**
```
sdk.dir=C\:\\Users\\你的用户名\\AppData\\Local\\Android\\Sdk
```

**macOS:**
```
sdk.dir=/Users/你的用户名/Library/Android/sdk
```

**Linux:**
```
sdk.dir=/home/你的用户名/Android/Sdk
```

### 2. 权限设置

安装后需要授予：
- ✅ 蓝牙权限
- ✅ 通知权限
- ✅ **无障碍权限**（重要！）

### 3. 无障碍权限设置

```
设置 → 无障碍 → 已下载的服务 → 蓝牙音乐控制 → 开启
```
