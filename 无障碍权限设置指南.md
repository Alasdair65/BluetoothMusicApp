# 🔐 无障碍权限设置指南

## 为什么需要无障碍权限？

由于 Apple Music 安卓版**不支持直接通过 Intent 播放特定歌曲**，我们需要使用无障碍服务来：

1. ✅ **搜索并播放 BGM 歌曲** - 模拟点击搜索按钮、输入歌曲名、点击播放
2. ✅ **检测 BGM 播放完成** - 监听播放状态变化
3. ✅ **自动切换到播放列表** - BGM 播放完后自动点击播放列表
4. ✅ **设置播放模式** - 模拟点击随机/循环按钮

**安全承诺**：无障碍服务仅在 Apple Music 应用运行时激活，不会收集任何个人信息。

---

## 📱 Android 16 设置步骤

### 方法一：首次启动时自动引导

1. 打开「蓝牙音乐自动化」应用
2. 授予蓝牙和通知权限
3. 点击「保存配置并启动监控」
4. 应用会提示需要无障碍权限
5. 点击「去设置」按钮

### 方法二：手动设置

**步骤 1：打开系统设置**
```
设置 → 无障碍 → 已下载的服务
```

**步骤 2：找到应用**
- 在列表中找到「蓝牙音乐控制」或「Bluetooth Music Control」

**步骤 3：开启服务**
- 点击开关，开启无障碍服务
- 系统会弹出警告对话框
- 点击「允许」或「确定」

**步骤 4：确认状态**
- 开关变为蓝色/绿色表示已启用
- 通知栏可能会显示无障碍服务已启动

## ⚠️ 常见问题

### Q1: 找不到「已下载的服务」？

**解决方案**：
1. 确保应用已安装
2. 重启手机
3. 在无障碍设置页面下拉刷新
4. 查看「已安装的应用」或「所有服务」分类

### Q2: 开启后立即关闭？

**可能原因**：
- 系统权限限制
- 电池优化干扰

**解决方案**：
1. 关闭电池优化（见下方）
2. 允许应用自启动
3. 锁定应用到最近任务

### Q3: 系统提示「可能有害」？

这是 Android 的标准安全警告，所有无障碍服务都会显示。

**安全说明**：
- 此警告是所有无障碍服务的标准提示
- 本应用是开源的，代码可审查
- 服务仅在 Apple Music 运行时激活
- 不会访问短信、通话、银行应用等敏感内容

**继续操作**：
1. 阅读警告内容
2. 点击「我了解风险」或「仍然允许」
3. 开启服务

## 🔋 关闭电池优化（重要）

为确保后台服务正常运行，需要关闭电池优化：

### 通用方法
```
设置 → 应用 → 蓝牙音乐自动化 → 电池 → 允许后台活动
```

### 小米手机
```
设置 → 应用设置 → 应用管理 → 蓝牙音乐自动化 → 省电策略 → 无限制
```

### 华为手机
```
设置 → 电池 → 应用启动管理 → 蓝牙音乐自动化 → 关闭「自动管理」→ 开启「允许后台活动」
```

### OPPO 手机
```
设置 → 电池 → 更多设置 → 应用耗电管理 → 蓝牙音乐自动化 → 允许完全后台行为
```

## ✅ 验证无障碍服务是否正常工作

### 方法 1：检查通知栏
- 开启服务后，通知栏可能显示「无障碍服务已启动」

### 方法 2：查看应用状态
- 打开「蓝牙音乐自动化」应用
- 查看是否有「无障碍服务已启用」提示

### 方法 3：测试功能
1. 配置好 BGM 歌曲和播放列表
2. 断开并重新连接蓝牙
3. 观察是否自动播放 BGM 并切换到播放列表

### 方法 4：查看日志
```bash
# 通过 ADB 查看日志
adb logcat | grep MusicAccessibilityService
```

## 🔄 重新启用服务

如果服务意外停止：

1. 打开系统设置 → 无障碍
2. 找到「蓝牙音乐控制」
3. 关闭开关，等待 3 秒
4. 重新开启开关
