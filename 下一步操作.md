## 🚀 下一步操作

### 1. 在本地构建项目

**你需要：**
- Android Studio（推荐）或
- JDK 11+ + Android SDK + 命令行工具

**步骤：**

```bash
# 1. 编辑 local.properties，设置你的 SDK 路径
# macOS: sdk.dir=/Users/你的用户名/Library/Android/sdk
# Windows: sdk.dir=C:\\Users\\你的用户名\\AppData\\Local\\Android\\Sdk
# Linux: sdk.dir=/home/你的用户名/Android/Sdk

# 2. 使用 Android Studio 打开项目
# File → Open → 选择 BluetoothMusicApp 文件夹

# 3. 等待 Gradle 同步完成

# 4. 构建 APK
# Build → Build APK(s)

# 5. APK 位置：
# app/build/outputs/apk/debug/app-debug.apk
```

### 2. 安装到手机

```bash
# 方法 A：通过 ADB
adb install app/build/outputs/apk/debug/app-debug.apk

# 方法 B：直接传输 APK 到手机，手动安装
```

### 3. 配置和使用

1. 打开应用，授予所有权限
2. 选择你的蓝牙设备
3. 选择 Apple Music
4. 设置 BGM 歌曲（可选）
5. 选择播放模式
6. 输入歌单名称
7. 保存配置
8. 断开并重新连接蓝牙测试

## 🔧 可能的调整

### 如果 Apple Music 自动化效果不理想

由于 Apple Music 安卓版的限制，可能需要调整：

1. **改用 Spotify**：对自动化支持更好
2. **使用 Intent 直接启动播放列表**：需要研究 Apple Music 的 Intent 格式
3. **结合 Tasker/MacroDroid**：用自动化应用补充 UI 模拟

### 如果需要更多功能

可以添加：

- 多个配置场景（车载/耳机/音箱）
